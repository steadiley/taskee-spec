# 制作アプリ
## タスク管理 + タイムロガーWEBアプリ

# 解消したい悩み(ユースケース)
- やることリスト(markdown) + そのタスクにかかった時間(clockify)を一つにしたい
- 複数のアプリを起動したくない
- 日毎に管理したい
- タスクごとの使用時間を図りたい
- 記録を保存して、可視化したい

# MVP (Minimum Viable Product)
- 日やることリスト (Today's Todo)を作成できる
- 各タスクで要した時間を記録できる
- グラフなどでタスクに要した時間を可視化できる

# 検証の方法について
- 静的解析
- 自動テスト
- CI/CD

# 技術的な制約
- CSSライブラリは使用しない
- Vue3を使用する
- APIサーバは用意しない

# 使用想定技術
- Vue3(TypeScript)
- firebase(firestore)
- SCSS

# ファイル管理方法
- 仕様書とソースコードはリポジトリを分けて管理
  - taskee-spec(仕様書類)
  - taskee(ソースコード)

# 開発場のルール
- 作業は必ずブランチを作成し、PRベースでの共有とする
